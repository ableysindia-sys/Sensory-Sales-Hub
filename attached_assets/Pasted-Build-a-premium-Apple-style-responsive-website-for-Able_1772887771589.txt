Build a premium Apple-style responsive website for Abley’s Rehab using React + Tailwind CSS. The visual language should feel minimal, elegant, modern, spacious, high-trust, and product-led — similar to Apple product pages, but adapted for professional therapy and rehabilitation equipment.

Brand setup
	•	Website name: Abley’s Rehab
	•	Hero / primary color: #4A53A0
	•	Secondary colors: white, light grey, charcoal black
	•	Accent use should be subtle and premium
	•	Avoid loud or toy-like design
	•	Tone should feel professional, institutional, polished, engineered, and warm
	•	Add soft shadows, large clean typography, lots of whitespace, rounded sections, premium cards, subtle gradients, and refined iconography

Business positioning

Abley’s Rehab sells professional therapy tools for experts such as:
	•	Occupational therapists
	•	Rehab centres
	•	Special schools
	•	Hospitals
	•	Clinics
	•	Sensory gyms
	•	Institutions

This is a B2B-first website with ecommerce capabilities, with clear options for:
	•	direct browsing of categories
	•	bulk order enquiries
	•	tool customization enquiries
	•	future scalability into ecommerce catalog/product pages

Brand statement

Use the motto prominently in the hero or brand section:

“When experts are equipped with professional tools, magic happens.”

Current and future structure

Right now the main focus is OT tools, but the website should visually communicate that the platform will expand into:
	•	Sensory Room
	•	Physiotherapy
	•	Speech Tools

Create the architecture in a way that supports future expansion.
