Update the current Abley’s Rehab website.
Right now it is behaving like a landing page with enquiry form, but the requirement is a B2B ecommerce catalogue website.

Keep the current design style, colours (#4A53A0, white, grey, black), and branding, but modify the structure so the site functions like a product catalogue with ecommerce capability.
